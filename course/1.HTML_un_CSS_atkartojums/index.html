Kā strukturēt HTML

Publicēts 19.08.2025 • #HTML

Semantiski tagi (header, nav, section, article, aside, footer) palīdz lasāmībai un piekļūstamībai.

CSS selektori praksē

• #CSS

  • Grupu atlase: .klase .bērns
  • Tiešais bērns: .konteiners > .elements
  • Blakus brālis: h2 + p
  • Attribute: a[target="_blank"], input[type="email"]
  • Stāvokļi: :hover, :focus, :disabled
  • Strukturālie: :nth-child(2n), :first-child

Uzdevums: Pievieno JavaScript

Šajā mapē jau ir fails script.js. Tavs uzdevums – pievienot šo failu lapai, lai JavaScript tiktu ielādēts un izpildīts.

Soļi

  1. Atver šo failu: index.html.
  2. Pievieno <script> elementu ar atsauci uz script.js.
  3. Saglabā un pārlādē lapu pārlūkā.

Kas tieši jādara?

Pirms </body> taga ievieto sekojošo:

<script src="script.js"></script>

Pēc pievienošanas, ielādējot lapu, būtu jāredz paziņojums no script.js.
